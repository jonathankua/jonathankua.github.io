Software Tools

  • TCP Experiment Automation Controlled Using Python (TEACUP)
    A Python-based software for orchestrating and automating TCP performance experiments in a dedicated, specially-constructed physical testbed. TEACUP was originally developed at the Centre for Advanced Internet Architectures at Swinburne University of Technology. Starting with TEACUP v1.0, development is community supported and publicly hosted on SourceForge. Please see the TEACUP project page for more information.

  • R-SIFTR Library
    A collection of R functions to simplify working with FreeBSD SIFTR log files. Current version v0.1, released on 1 Oct 2014. The library can be downloaded here.
